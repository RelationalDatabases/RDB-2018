Objectives

This lab will firstly introduce you to MySQL. It will show you how to setup MySQL on your own laptop.

Setup

Instructions for setting up MySQL

First you must have an oracle account. Go to - http://Oracle.com/ and sign up for an oracle account. Next go to - https://dev.mysql.com/downloads/mysql/ Scroll down the page and choose your operating system from the dropdown box.

If you have a windows machine click on the Go to Download Page button that appears in the first image for MYSQL Installer for Windows,

Then click on the either the web-community download or the full installer download as seen in the image shown below:

There is documentation to help with your install process.

  • https://dev.mysql.com/doc/refman/8.0/en/mysql-installer.html

The MySQL installer will bring you through the process, you need to choose the full install so all products are installed. Make sure you setup a root user and make a note of the password you set as you will need it frequently during this semester.

Workbench should start once the installation is complete.

You should see the instance that was setup by default during the install. Double click on it now and using the password you set you can login.

Once you are logged in you will see the following screen:

Download and install MySQL sample database

Download the following file:

-classic models database

Extract the files onto your file system.

In workbench choose File-> Open SQL Script , choose the file that you have just extracted from the zip, it is named mysqlsampledatabase.sql

It is now viewable in the workbench environment.

To execute the SQL script you click Execute button shown by the lightening icon

Right click inside the Schema panel and click Refresh All button to update the panel.

You should now see the classic model database is loaded successfully into MySQL database server.
